# Clots: Hem Empire

Полноценная стратегическая игра о кровяной империи. Вы — разумная система крови,
которая строит собственную цитадель в сосудистой сети, управляет ресурсами,
исследует сектора и выдерживает атаки иммунной системы.

## Суть проекта
- **Идея:** RPG/стратегия, где игрок управляет не героем, а самой системой крови.
- **Цель:** стабилизировать карту кровотока, укрепить ядро и выжить при росте угрозы.
- **Механики:** управление ресурсами, пассивная добыча, исследование карты, бои,
  модульные улучшения, журнал событий.

## Быстрый старт
```bash
npm install
npm run dev
```

## Основные элементы
- **Цитадель крови:** ключевые показатели (маскировка, угроза, целостность).
- **Карта кровотока:** цепочка секторов с разными типами событий.
- **Боевой контур:** пошаговые столкновения с иммунными угрозами.
- **Модули:** улучшения ядра и пассивные эффекты.
